================================================================================
LECTURE À FAIRE
================================================================================

Si la MAE et la RMSE augmentent fortement dans les segments chers, cela confirme que le modèle prédit moins bien les logements haut de gamme.
Si l'erreur moyenne est négative dans les segments chers, cela signifie que le modèle sous-estime ces logements.
Si le taux de sous-estimation dépasse largement 50 %, cela signifie que le modèle prédit trop souvent un prix inférieur au prix réel dans ce segment.

Pistes d'amélioration à tester ensuite :
- pondération plus forte des logements chers pendant l'entraînement ;
- Quantile Loss avec alpha > 0.5 pour limiter la sous-estimation ;
- calibration des prédictions sur validation ;
- ajout de variables textuelles ou visuelles pour mieux capter le standing ;
- modèle spécialisé pour logements premium, mais sans utiliser le vrai prix au moment de la prédiction.
